What is Subtitle Adding Script?

Premiere Pro Subtitle Adding Script is

  • A python tool for premiere pro `.SRT` file syntax and formatting, are used for repetitive, and a lot of school video editing.
  • This tool replaced input subtitles directly into software nor easily messed up `.SRT` file.

  • Speeding up formatting and finalizing subtitles greatly, there is a minor problem that if the line is too long that needs manually fix.
